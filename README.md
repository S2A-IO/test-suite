# Setup instructions
Install packages

```
npm install
```

# Running tests

## Run all tests

```
npm run test:all -- --src=<path-to-project-root> --app=<name-of-app>
```
## Run some tests

```
npm run test -- --src=<path-to-project-root> --app=<name-of-app> <path-to-a-single-test-or-path-pattern>
```
